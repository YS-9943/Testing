Git- It is a version control system.
version control - software that helps us to track changes to a file over a period of time.

Git Project
1 open command line then move to the folder
2 git init - git repository initialize/start
3 to start saving  changes - git add filename -> git add .
4 git commit -m "Header added"

5 to check if file need to be commited -  git status
6 to check all commits - git log --oneline

7 to visit any particular commit-  git checkout id
8 to visit back at current position - git checkout master

9 to connect local file to remote file -  git remote add origin link
10 to push code to remote file- git push origin master
